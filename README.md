Dashboard interativo para monitoramento epidemiológico de Dengue em Araras-SP (2019-2024). Desenvolvido para o Projeto de Extensão II - UNINTER.
# 🦟 Monitoramento de Dengue - Araras/SP

Este projeto é uma ferramenta de **Business Intelligence (BI)** desenvolvida para a disciplina de **Projeto de Extensão II** do curso de Ciência de Dados da **UNINTER**.

## 🎯 Objetivo
O objetivo principal é fornecer uma interface visual e interativa para gestores de saúde e cidadãos de Araras-SP, facilitando a identificação de bairros críticos e a análise da sazonalidade da Dengue na região.

## 🚀 Tecnologias Utilizadas
* **Linguagem:** Python 3.x
* **Framework Web:** Streamlit
* **Visualização de Dados:** Plotly Express
* **Manipulação de Dados:** Pandas

## 📊 Funcionalidades
* **Filtro Temporal:** Análise por anos específicos ou série histórica completa ("Todos").
* **Métricas em Tempo Real:** Total de casos, bairro mais afetado e região crítica.
* **Análise Geográfica:** Ranking dos 10 bairros com mais notificações.
* **Sazonalidade:** Gráfico de área mostrando os meses de pico (Abril/Maio).
* **Mapa de Calor:** Visualização da persistência da doença nos bairros ao longo dos anos.
